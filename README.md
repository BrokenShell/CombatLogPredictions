# Combat Logs
Generate combat logs for machine learning project.


## CombatUnit
- Name
- Class
- Level
- Health
- Damage
- Offence
- Defense


## Simple D20 system
### Attack
d20 + offence bonus

### Defend
d20 + defense bonus

### Hit
attack > defend

### Damage
d(dam_dice) for each level of the attacker
